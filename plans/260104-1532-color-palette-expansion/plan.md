# Color Palette Expansion Plan

**Date**: 2026-01-04
**Priority**: Medium
**Status**: Planning

## Overview

Mở rộng color palette từ cam + đen sang **cam + đen + trắng + xanh lá** để giảm sự áp đảo của màu cam trong giao diện.

## Current State

- Primary: Coral/Orange `#D97755`
- Background: Dark `#1A1A1A`
- Cards: `#212121`
- Text: `#F2F2F2`
- Success (limited): `#22C55E`

## Target State

Balanced 4-color palette với semantic meaning rõ ràng.
